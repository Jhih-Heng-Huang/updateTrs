This program can update the content of .trs file when you modified the content of .text file.

Steps:

1. Write down each text files with the full path in text.lst.
The following two lines are the example in text.lst:
	/home/albert/work/corpus/temp/_Tsai_Feng_An_20-20121111-1021214-1030215.text
	/home/albert/work/corpus/temp/_Tsai_Feng_An_30_131105-1030101-0119.text

2. Run text2trs.pl with the following command in the linux:
	perl text2trs.pl
