The program in this folder can generate a ".text" file from its source ".trs" file.


Steps:

1. Write down each trs files with the full path in trs.lst.
The following two lines are the example in trs.lst:
	/home/albert/work/corpus/temp/_Tsai_Feng_An_20-20121111-1021214-1030215.trs
	/home/albert/work/corpus/temp/_Tsai_Feng_An_30_131105-1030101-0119.trs

2. Run trs2text.pl with the following command in the linux:
	perl trs2text.pl
